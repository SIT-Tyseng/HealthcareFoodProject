# Image Reviewer

## Installation
To install `node` and `npm`:
```
sudo apt update
sudo apt install -y curl
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install --lts
node --version
npm --version
```

To start running the app:
```
In new terminal 1:
node server.js 
```
```
In new terminal 2:
npm start 
```
